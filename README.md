## 가이드 라인
* 도커는 실제로 직접 사용하지 않고, 쿠버네티스로 관리합니다.
* 도커에서는 이미지 만드는 법이랑 배포하는 법만 아셔도 무방합니다.

* 빠르게 필요한 부분만 보실거면 
	- `4. 도커 명령어`
	- `5. 도커 파일`
	- `6. 도커 이미지 배포`
만 보셔도 됩니다.
